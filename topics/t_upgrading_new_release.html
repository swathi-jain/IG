Upgrading Rational Focal Point

By using IBM® Installation Manager, you can upgrade to the latest release of IBM Rational® Focal Point™.

Before you begin

  • Back up the database and any custom business rules. For more information, see Upgrading from earlier versions.
  • Verify that your version of the license server is supported for the release. If necessary, upgrade to the supported version of IBM Rational License Server.
  • Verify that your environment meets the system requirements. If you are using a version of software that is no longer supported, you must upgrade to a version that is compatible with Rational Focal Point. For information about upgrading a software product, see the documentation for that product.
  • If you are using an IBM DB2® or Oracle database, make sure that you created the database tables with the specified table space names for Rational Focal Point.
  • Obtain the installation media or download the installation package. You can install Rational Focal Point from the installation media, or from a download package that is available on the IBM Passport Advantage® website or IBM Fix Central.

Procedure

  1. In IBM Installation Manager, click File > Preferences.
  2. On the Repositories page, click Add Repository and specify the repository location as Rational Focal Point package location/disk1.
  3. Click OK.
  4. Click Update and select Rational Focal Point.
  5. If an older version of IBM Installation Manager is installed, proceed with the update of IBM Installation Manager and then update Rational Focal Point.
  6. Read the license agreement. If you agree with its terms, accept the license agreement and click Next.
  7. On the Features page, click Next.
  8. On the IBM Rational Focal Point Application Server page, select the application server.
    • If you select Apache Tomcat application server, specify the application server details.
      1. Select the application server and click Browse Server Home to specify the location of the application server home path.
        Note: While upgrading, if the application server version is no longer supported by Rational Focal Point, point to the home path of the new supported version of the application server.
      2. Specify the memory usage limit for the application server.
        Important: Make sure that the required memory space is available on the computer. If memory space is insufficient, the application server might not work properly or start.
      3. Select the default locale for Rational Focal Point. Continue to step 9.
    • If you select IBM WebSphere® Application Server, you must manually set up and start the server after the installation. Proceed to step 10.
  9. On the Database Type Selection and Configuration page, specify the database details.
    1. Select the application database.
    2. Type the host name/IP address, port, and name of the database.
      Note: If you are using an Oracle database, specify the database SID instead of the database name.
      Table 1. Parameter value examples
      Database Parameter value examples
      DB2
      • Database Server Host Name/IP:127.0.0.1
      • Database Port:50000
      • Database Name: FOCALDB
      Oracle
      • Database Server Host Name/IP:127.0.0.1
      • Database Port:1521
      • Database Name: ORCL10g
      PostgreSQL
      • Database Server Host Name/IP:127.0.0.1
      • Database Port:5432
      • Database Name: focalpoint
    3. Type the user name and the password for the database.
      • For DB2 and PostgreSQL databases, the database connection parameters are the same settings that you specified when you installed the database.
      • For Oracle databases, specify the user name and the password that you created while creating the Rational Focal Point schemas.
  10. On the IBM Rational Focal Point License Information page, specify the license information.
    • Select I will supply license information later to proceed with the installation. You must specify the license information when you log in to Rational Focal Point.
    • To specify the license server, select A Rational License Server. Type the location of the license server in the following format: port@host. For example: 27000@licenseserver. For information about the port number that must be specified, see the Rational License Key server documentation.
    • If you have a license file, copy the file to a folder on your computer and type the path of the license file.
  11. Click Next.
  12. Click Update.
  13. A message confirms the completion of the process.
    • To verify the installation, click View Log File. The installation log file for the current session opens in a new window. To close the log file window and return to the Install wizard, click Close.
    • To close the Install wizard without verifying the installation, click Finish.

What to do next

  • If you are using WebSphere Application Server, you must manually set up and start the server. For more information, see Setting up WebSphere Application Server.
  • After you upgrade to the latest release of Rational Focal Point, migrate the required data. For more information, see Migrating data from earlier versions of Rational Focal Point
  • The database verifier feature is available from Rational Focal Point version 6.5.1 release. After you migrate the data, perform the database verifier operation for verifying the database after the upgrade. For more information, see Verifying the database
