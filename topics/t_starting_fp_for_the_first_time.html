Starting Rational Focal Point for the first time

Show Me
After you install IBM® Rational® Focal Point™, start the program and change the administrator password.

Procedure

  1. Start the application server. Verify that Rational Focal Point is running.
  2. In a web browser, type the URL of the Rational Focal Point login page in this format: http://localhost:port number of the application server/fp/servlet/Login. For example: http://localhost:8080/fp/servlet/Login.
  3. On the login page, in the User Name field, type admin. In the Password field, type focalpoint.
  4. Click Log In.
  5. Optional: If you did not specified the license information during the installation, you must now specify the Rational license server details in the following format: port@host. For example: 19353@licenseserver.
  6. On the home page, click Preferences.
  7. Click Profile/Password Settings and change your password.
  8. Optional: You can change your user name on the Users/Manage Users page, which is on the navigation bar.

What to do next

For Rational Focal Point version 6.5, 6.5.0.1, and 6.5.0.2, the help contents are not installed with the product. After you install or upgrade Rational Focal Point, you must download the help from the IBM Local Help System Updater website. For more information, see Downloading Rational Focal Point help
