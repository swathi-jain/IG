User account and permission requirements

To install Rational® Focal Point™, your user ID must have specific characteristics and permissions.
Before you install the product, verify that your user ID meets the following criteria:
  • Your user ID must not contain double-byte characters.
  • For Microsoft Windows operating systems: Your user account must have certain access rights that depend on the version of your operating system. Your user ID must belong to the Administrators group.
  • For Linux: You must have the access and permissions corresponding to the root user.
