Installing the licenses for Rational Focal Point

To use IBM® Rational® Focal Point™, install the Rational License Key Server and the Rational Focal Point licenses. Then, either during the installation or when you start the product for the first time, configure the product to point to the license server.

Before you begin

You must know which version of Rational License Server to use for your instance of Rational Focal Point. For more information, see License requirements.

Procedure

  1. Install the Rational License Key Server.

    Rational License Server is included with the Rational Focal Point installation media. You can also retrieve the latest version of Rational License Server through your Passport Advantage® account or at the Rational Download and Licensing Center.

    For information about installing and configuring the license server, see the following topics in the Rational License Server information center:
    • For Windows systems: Configuring a license server for Windows
    • For AIX® or Linux systems: Configuring a license server on UNIX
  2. Request and install the Rational Focal Point licenses in the Rational license server.
    1. Copy the license file to a directory on the license server. An example of a license file is LicAuthFixedtermUserbased.dat.
    2. Start LMTools. The default location is C:\Program Files\IBM\RationalRLKS\common\lmtools.exe.
    3. Click Config Services.
    4. Click the Browse button next to Path to license file input file, browse to the license file that you copied, and then click Save Service.
    5. In the Services panel, start the FLEXlm License Manager service.
    6. Click the Server Status tab and click Perform Status Enquiry. A status report of the available licenses opens.
  3. Configure Rational Focal Point to point to the license server.
    • If you are using Apache Tomcat server, you can specify the license information during the installation or when you start the product for the first time.
    • If you are using IBM WebSphere® Application Server, specify the license information when you set up Rational Focal Point to work with WebSphere Application Server.
