Installing Rational Focal Point

Follow these instructions to install IBM® Rational® Focal Point™.

Before you begin

  • Obtain the installation media or download the installation package. You can install Rational Focal Point from the installation media, or from a download package that is available on the IBM Passport Advantage® website.
  • Verify that your computer meets the system requirements, and complete the preinstallation tasks.
  • Make sure that you have administrator access to the server.
  • Make sure that the application server is connected to the database and license servers.
  • Make sure that your computer is connected to the Internet.
  • If you are using an Oracle or IBM DB2® database, ensure that you have created the database tables with the specified table space names for Rational Focal Point.

Procedure

To install Rational Focal Point on the Windows or Linux operating systems:

  1. If you are using the installation media, insert the media into the disk drive. If you are using the download package, extract the files from the downloaded .zip package to a temporary directory.
  2. Open the launchpad program to start the package installation:
    • On Windows systems: Click launchpad.exe.
    • On Linux systems: Click launchpad.sh.
    Note: If the launchpad program fails to start the installation, then run the install file from the platform specify install folder. For example: InstallerImage_Win32, IMImage_LinuxGTKX86 folder.
  3. Click Install IBM Rational Focal Point.
  4. On the IBM Installation Manager Start page, click Install. The Install Packages page lists all of the packages in the repositories that Installation Manager searched. If two versions of a package are discovered, only the most recent, or prescribed, version of the package is displayed. If Installation Manager is not installed on your computer, that product is listed on the Install Packages page with Rational Focal Point.
  5. To display all versions of the packages that Installation Manager finds, click Show all versions.
    Note: To search the predefined IBM update repository locations for the installed packages, make sure that Search the linked repositories during installation and updates is selected on the Repositories preference page. This preference is selected by default.
  6. Select Rational Focal Point, the required version, and Installation Manager, if necessary. Click Next.
  7. Read the license agreement. If you agree with its terms, accept the license agreement and click Next.
  8. Either accept the default installation directory for Installation Manager or click Browse and specify a different installation location. Click Next.
  9. Either accept the default installation directory for Rational Focal Point or click Browse and specify a different installation location. Click Next.
  10. On the IBM Rational Focal Point Application Server page, select the application server.
    • If you select Apache Tomcat application server, specify the application server details.
      1. Select the application server and click Browse Server Home to specify the location of the application server home path.
      2. Specify the memory usage limit for the application server.
        Important: Ensure that the required memory space is available on the computer. If memory space is insufficient, the application server might not work properly or start.
      3. Select the default locale for Rational Focal Point. Continue to step 11.
    • If you select IBM WebSphere® Application Server, you must manually set up and start the server after the installation. Proceed to step 13.
  11. On the Database Type Selection and Configuration page, specify the database details.
    1. Select the application database.
    2. Type the host name/IP address, port, and name of the database.
      Note: If you are using Oracle, specify the database SID instead of the database name.
      Table 1. Parameter value examples
      Database Parameter value examples
      IBM DB2
      • Database Server Host Name/IP:127.0.0.1
      • Database Port:50000
      • Database Name: FOCALDB
      Oracle
      • Database Server Host Name/IP:127.0.0.1
      • Database Port:1521
      • Database Name: ORCL10g
      PostgreSQL
      • Database Server Host Name/IP:127.0.0.1
      • Database Port:5432
      • Database Name: focalpoint
    3. Type the user name and the password for the database.
      • For DB2 and PostgreSQL databases, the database connection parameters are the same settings that you specified when you installed the database.
      • For an Oracle database, specify the user name and the password that you used when you created the Rational Focal Point schemas.
  12. On the IBM Rational Focal Point License Information page, specify the license information.
    • Select I will supply license information later to proceed with the installation. You must specify the license information when you log in to Rational Focal Point.
    • To specify the license server, select A Rational License Server. Type the location of the license server in the following format: port@host. For example: 27000@licenseserver. For information about the port number that must be specified, see the Rational License Key server documentation.
    • If you have a license file, copy the file to a folder on your computer and type the path of the license file.
  13. Click Next.
  14. Click Install.
  15. A message confirms the completion of the process.
    • To verify the installation, click View Log File. The installation log file for the current session opens in a new window. To close the log file window and return to the Install wizard, click Close.
    • To close the Install wizard without verifying the installation, click Finish.

What to do next

  • If you are using WebSphere Application Server, you must manually set up and start the server. For more information, see Setting up WebSphere Application Server.
  • If you are migrating from an earlier version of Rational Focal Point, you can import data from that version. For more information, see Migrating data from earlier versions of Rational Focal Point.
  • For Rational Focal Point version 6.5, 6.5.0.1, and 6.5.0.2, the help contents are not installed with the product. After you install or upgrade Rational Focal Point, you must download the help from the IBM Local Help System Updater website. For more information, see Downloading Rational Focal Point help
