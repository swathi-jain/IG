Starting the Rational Focal Point application and the components

Before you begin

  1. Deploy the Rational® Focal Point™ application
  2. Deploy the financial feature component
  3. Deploy the Jazz Web UI feature component
  4. Deploy the help system

Procedure

  1. On the IBM WebSphere Integrated Solutions Console, click Application > Enterprise Applications.
  2. Select the following check boxes: fp_war, Financier_war, FinancierRest_war, jazzui_war, and fphelp_war.
  3. Click Start. A green arrow indicates that the application started.

What to do next

  • Specify where to save your Rational Focal Point logs. For more information, see Rational Focal Point log files.
  • The administrator must start Rational Focal Point and change the administrator password. For more information, see Starting Rational Focal Point for the first time.
  • Enabling Java 2 security for Rational Focal Point on WebSphere Application Server
