Rational Focal Point log files

Depending on which application server you use, the log files for IBM® Rational® Focal Point™ are written to different locations.

Log file locations for Apache Tomcat

If you use Apache Tomcat server, log files such as fp.date.log and catalina.out are written to tomcat installation directory/logs.

Log file locations for WebSphere Application Server

  • If you are using WebSphere® Application Server, the log files are written to WAS installation directory\profiles\servername\logs\ by default.
    Tip: To make it easier for IBM Software Support to troubleshoot your installation, keep the log files in the default location.
    To modify the location of the Rational Focal Point log files in WebSphere Application Server:
    • Open the fp-logging.properties file in the following directory: WAS installation directory\profiles\servername\installedApps\cellname\fp_war.ear\fp.war\WEB-INF\classes
    • Modify the following entry to specify a different location for logs: java.util.logging.FileHandler.pattern= WAS installation directory\profiles\servername\logs\fp.log
    • Restart the application server.
