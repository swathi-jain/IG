Installing and upgrading Rational Focal Point silently

You can install or upgrade IBM® Rational® Focal Point™ from the command line by using response files rather than using the IBM Installation Manager graphical user interface.

Before you begin

  • You can install Rational Focal Point from the installation media or from a download package that is on the IBM Passport Advantage® site. Before you begin the installation, obtain the installation media or download the installation package.
  • Verify that your computer meets the system requirements, and complete the preinstallation tasks.
  • Make sure that you have administrator access to the server.
  • Make sure that the application server is connected to the database and license servers.
  • Make sure that your computer is connected to the Internet.

About this task

For complete details about silent installation, see the Installation Manager information center.

Procedure

To install or upgrade IBM Installation Manager and Rational Focal Point silently:

  1. Create a response file that contains the commands and inputs for the installation or the upgrade process. For instructions to create and use a response file, see Creating a response file manually.
  2. Open a command prompt and change to the following directory: InstallerImage_platform. For example: cd C:\FocalPoint\disk1\InstallerImage_win32_win32_x86
  3. Enter the installation command.
    • On Windows systems: installc.exe -acceptLicense input response_file_path_and_name -log log_file_path_and_name
    • On Linux systems:./installc -acceptLicense input response_file_path_and_name -log log_file_path_and_name
    Note: The commands may change based on the version of Installation Manager. For more information, see Installation Manager information center

What to do next

  • If you are using IBM WebSphere® Application Server, you must manually set up and start the server. For more information, see Setting up WebSphere Application Server.
  • If you are migrating from an earlier version of Rational Focal Point, you can import your data from the earlier version. For more information, see Migrating data from earlier versions of Rational Focal Point.
  • For Rational Focal Point versions 6.5, 6.5.0.1, and 6.5.0.2, the help system is not installed with the product. After you install or upgrade the product, you must download the help contents from the IBM Local Help System Updater website. For more information, see Downloading Rational Focal Point help
  • Creating a response file manually
    To silently install or upgrade Rational Focal Point, you can use a response file that contains the data required to install the product.
  • Sample response file for installing and upgrading
    You can use the sample response file in this topic to install Rational Focal Point. You must modify the parameter values for your installation environment.
