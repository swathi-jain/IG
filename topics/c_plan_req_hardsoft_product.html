System requirements

Verify that your hardware and software meet or exceed the minimum requirements to install Rational® Focal Point™.

The following table shows the minimum system requirements to install and use Rational Focal Point. Check the compatibility between the operating systems, browsers, JREs, databases and the application servers before you install Rational Focal Point.

Table 1. System requirements to install Rational Focal Point
Item Requirements
Hardware
  • Memory: 1 GB minimum, 8 GB recommended
  • Disk space: 10 GB minimum, 80 GB recommended
  • Display: Minimum resolution 1024 x 768 on a client computer
Platform One of the following operating systems must be installed:
  • Microsoft Windows Server 2003 Enterprise Edition, SP1, SP2 (32-bit x86, 64-bit x64)
  • Microsoft Windows Server 2003 Standard Edition, SP1, SP2 (32-bit x86, 64-bit x64)
  • Microsoft Windows Server 2008 Enterprise Edition, SP2 (32-bit x86, 64-bit x64)
  • Microsoft Windows Server 2008 Standard Edition, SP2 (32-bit x86, 64-bit x64)
  • Microsoft Windows Server 2008 Enterprise Edition SP1 R2 (64-bit x64)
  • Microsoft Windows Server 2008 Standard Edition SP1 R2 (64-bit x64)
  • Red Hat Enterprise Linux 5.0 Advance Platform (32-bit x86, 64-bit x64)
  • Red Hat Enterprise Linux 5.0 Desktop editions (32-bit x86, 64-bit x64)
  • Red Hat Enterprise Linux 6.0 Server edition (32-bit x86, 64-bit x64)
  • Oracle Solaris 9, 10 or 11 SPARC
Supported platforms in the client computer to access Rational Focal Point:
  • Microsoft Windows XP
  • Microsoft Windows Vista
  • MicrosoftWindows 7
Application server One of the following application servers must be installed:
  • From Rational Focal Point version 6.5.2: Apache Tomcat 6.0
  • For Rational Focal Point version 6.5, 6.5.0.1, 6.5.0.2, 6.5.1 and 6.5.1.1: Apache Tomcat 5.5.33. The Apache Tomcat application server that was provided with Rational Focal Point in versions earlier than 6.4.1 is not bundled with the releases of the version 6.5 package.
  • IBM® WebSphere® Application Server 7.0.0.17.
Important: If you set up a clustered environment, make sure that the system time on the different servers is synchronized and differs by less than one second.
Relational database One of the following databases must be installed:
  • IBM DB2® Enterprise Server Edition 9.7
  • Oracle Database 10g Standard Edition Release 2
  • Oracle Database 10g Enterprise Edition Release 2
  • Oracle Database 11g Enterprise Edition Release 2
  • PostgreSQL 8.3
  • PostgreSQL 8.4
Java™ SE Runtime Environment (JRE)
  • Server:
    • Oracle JRE 6
    • IBM JRE6 with SR9
  • Client::
    • Oracle JRE 1.6
    • Oracle JRE 1.7 Update 17 (32-bit) (Tested browsers: FireFox 10.0.0.2 (32-bit), Internet Explorer 8 (32-bit) and Internet Explorer 9 (32-bit)
    • IBM JRE 1.6 with SR9 or higher versions
Web browser You can use these browsers to log in to Rational Focal Point:
  • Mozilla Firefox 9, 10 ESR
  • Microsoft Internet Explorer 7, 8, or 9
  • Google Chrome 17
Licensing server
  • For Rational Focal Point version 6.5, 6.5.0.1, 6.5.0.2 and 6.5.1: IBM Rational Common License Server 8.1.1
  • For Rational Focal Point version 6.5.1 and later: Rational Common License Server 8.1.2 is also supported.
For more information, see Managing licenses with Rational Common Licensing.
Installation Manager prerequisite
  • On Linux: The GNU Image Manipulation Program Toolkit (GTK+), Version 2.8.11, or later
If IBM Installation Manager causes installation issues, see the Installation Manager documentation and the technical support notes.
Note: For any platform, you may choose to use the unsupported platform but IBM will address defects or issues on the non supported platform only if the client can reproduce the error in a platform currently supported by IBM.
