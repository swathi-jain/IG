Installation Manager overview

IBM® Installation Manager installs, updates, and modifies packages. You can use it to manage the IBM applications, or packages, that it installs on your computer. With Installation Manager, you can install packages, track what you installed, determine what is available to install, and organize installation directories.

Installation Manager provides tools to update and modify packages, manage the licenses for your packages, and uninstall packages.

You can use the six wizards in Installation Manager to complete the following tasks:

  • Install a package by using the Install wizard. In the wizard, you can accept the defaults, or you can complete a custom installation. Before you begin the installation, you can view a complete summary of your selections. You can install one or more packages at a time.
  • Update packages by using the Update wizard. This wizard searches for available updates to packages that you installed. An update might be a released fix, a new feature, or a new version of the product. Details of the update are provided in the wizard. You can choose whether to apply an update.
  • Modify certain elements of an installed package by using the Modify wizard. During the first installation of a package, you select the features to install. Later, if you need other features, you can use the Modify wizard to add them to your package. You can also remove features and add or remove languages.
  • Set up licenses for your packages by using the Manage Licenses wizard. You can change your trial license to a full license, set up servers for floating licenses, and select the type of license for each package.
  • Revert a package to a previous version by using the Roll Back wizard.
  • Remove a package from your computer by using the Uninstall wizard. You can uninstall more than one package at a time.
Note: For Rational Focal Point version 6.5 releases, you can use IBM Installation Manager to only install, update, modify, and uninstall the product. With this version, IBM Installation Manager does not support other activities, such as managing licenses, and rolling back versions.
For more information about Installation Manager, see http://publib.boulder.ibm.com/infocenter/install/v1r4/index.jsp

For example: On Windows, click Start > Programs, right-click IBM Installation Manager. In the shortcut target field add -nl ar value.
  • If you use Hebrew locale, you must start the IBM Installation Manager by specifying the properties -nl en. IBM Installation Manager does not support Hebrew locale.
