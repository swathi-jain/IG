Planning the installation

When you plan an installation, you must consider several options and requirements. Use the installation road map to get started.
  • Installation road map
    The installation road map diagram shows the procedure to set up the environment and install IBM® Rational® Focal Point™. For more information about a task, click a related area in the diagram.
  • Installation Manager overview
    IBM Installation Manager installs, updates, and modifies packages. You can use it to manage the IBM applications, or packages, that it installs on your computer. With Installation Manager, you can install packages, track what you installed, determine what is available to install, and organize installation directories.
  • Installation requirements
    Before you install Rational Focal Point, make sure that you meet the requirements for hardware, software, server environment, operating systems, and other factors.
  • License requirements
    The licenses for IBM Rational Focal Point version 6.5 and later are managed through the Rational Common Licensing process. To use Rational Focal Point, install IBM Rational License Key Server and the Rational Focal Point licenses, and then configure the Rational Focal Point server to point to the license server.
  • Paths for installing Rational Focal Point behind a proxy server
    You can install Rational Focal Point on a computer that is behind a proxy server.
