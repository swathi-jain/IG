Post-installation tasks

After you install IBM® Rational® Focal Point™, you must complete other tasks, such as deploying the help and starting the server.
  • Starting the Apache Tomcat server
    After you install Rational Focal Point, you must start the Apache Tomcat server.
  • Migrating data from earlier versions of Rational Focal Point
    After you upgrade to the latest version, you can migrate the baselines and reports that you generated in earlier versions. To use your custom business rules, you must obtain the new compiled version from support and import that version.
