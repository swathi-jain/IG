Starting the Apache Tomcat server

After you install Rational® Focal Point™, you must start the Apache Tomcat server.

Procedure

To start the Apache Tomcat server from the command line:

  1. Change to the following directory: Apache Tomcat install directory\bin
    Note: If you are using a 32-bit JRE on a 64-bit operating system, set the flag FP_SERVER_64BIT to false in the fpenv file. On Windows, the file is available in Apache Tomcat install directory\bin\fpenv.bat and on Linux, Apache Tomcat install directory\bin\fpenv.sh.
  2. Enter the following command: startup.bat

    A separate Apache Tomcat console window opens. If you close this window, the server stops. Depending on your installation options, the server startup might take a few minutes. After the server is started, the startup time in millisecond is displayed in the Tomcat window. For example: Server startup in 203645 ms.

What to do next

  • Specify where to save your Rational Focal Point logs. For more information, see Rational Focal Point log files.
  • The administrator must start Rational Focal Point and change the administrator password. For more information, see Starting Rational Focal Point for the first time.
