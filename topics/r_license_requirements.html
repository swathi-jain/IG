License requirements

The licenses for IBM® Rational® Focal Point™ version 6.5 and later are managed through the Rational Common Licensing process. To use Rational Focal Point, install IBM Rational License Key Server and the Rational Focal Point licenses, and then configure the Rational Focal Point server to point to the license server.

Supported license servers

For information about Rational Common Licensing, see the Rational Support licensing page.

Table 1. Supported versions of the license server
Rational Focal Point release Supported license server
  • Earlier than version 6.5
  • IBM Telelogic License Server
  • 6.5
  • 6.5.0.1
  • 6.5.0.2
  • Rational License Key Server, version 8.1.1
  • 6.5.1 and later
  • Rational License Key Server, version 8.1.1
  • Rational License Key Server, version 8.1.2

Upgrading to the required version of Rational License server

  • If you are upgrading from earlier versions of Rational Focal Point to version 6.5 or later, you must migrate from IBM Telelogic License Server 2.0 to IBM Rational License Server. For more information, see Migrating from the Telelogic Licensing Server to Rational Common Licensing.
  • To upgrade to the latest version of Rational License Key Server, see Updating the Rational License Key Server.
