Uninstalling Rational Focal Point

To uninstall Rational® Focal Point™, use the Uninstall Packages option in IBM® Installation Manager. You can uninstall the packages from a single installation location.

Before you begin

Stop the server.

Procedure

  1. Log in to the system by using the same user account that you used to install the product packages.
  2. On the Start page of Installation Manager, click Uninstall Packages.
  3. On the Uninstall Packages page, select the Rational Focal Point product package.
  4. Click Next and then click Uninstall. When the process is finished, the Complete page is displayed.
  5. Click Finish to exit the wizard.

What to do next

The installer does not remove data that was created by users. You must delete this data manually. If you want to reinstall the product and point to this data, do not delete the data.
