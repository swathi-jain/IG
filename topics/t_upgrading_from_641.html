Upgrading from releases of Rational Focal Point version 6.4.1

If you installed a release of IBM® Rational® Focal Point™ version 6.4.1 and want to install the latest release, you can uninstall version 6.4.1 and install the latest release. You can also upgrade directly to the latest release.

Before you begin

  • IBM WebSphere® Application Server Community Edition is not supported by Rational Focal Point version 6.5 and later. You must use Apache Tomcat or WebSphere Application Server. For more information about the system requirements, see System requirements.
  • You must migrate from IBM Telelogic License Server 2.0 to IBM Rational Common License Server. For more information, see http://publib.boulder.ibm.com/infocenter/rational/v0r0m0/topic/com.ibm.rational.license.doc/topics/t_migrate_telelogic.html.

About this task

To upgrade from version 6.4.1 to the latest release, you must back up the database and some files. Then, you can either uninstall and reinstall the product, or upgrade the product.

Procedure

  1. Open your instance of Rational Focal Point and verify the view definitions on the Views page (Advanced > Verify Integrity > Views). Fix any broken view definitions that might be limitations for incoming link attributes, including views in inactive or archived workspaces.
  2. Note any custom business rules that you have used. Log on to Rational Focal Point and export the custom business rules.
    Important: After you install the latest version, for compatibility reasons, you must request the custom business rules from the IBM Rational Focal Point support team. Only the common business rules are included in the installation. The common business rules are ChoiceAttributeText, LinkByText, LinkInfo, LinkListAttributeSum, ListAttributeSum, MailImport, MatrixWriter, SetChoice, and TrafficLight. Incompatible business rules are ignored.
  3. Back up the database.
  4. Stop the application server.
  5. To uninstall and then reinstall Rational Focal Point, useIBM Installation Manager. For more information, see Uninstalling Rational Focal Point and Installing Rational Focal Point.
  6. To upgrade directly to the latest release of Rational Focal Point, see Upgrading Rational Focal Point.

What to do next

After you upgrade to the latest release of Rational Focal Point, migrate the required data. For more information, see Migrating data from earlier versions of Rational Focal Point.
